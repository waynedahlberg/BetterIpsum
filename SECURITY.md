# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.9.x   | Yes       |

## Reporting a Vulnerability

Please do not file a public GitHub issue for security vulnerabilities.

Email **wayne@waynedahlberg.com** with:
- A description of the issue
- Steps to reproduce
- Any relevant logs or screenshots

You'll receive a response within 7 days. If the report is confirmed, a fix will be released as soon as possible and credited to you in the changelog (unless you prefer to remain anonymous).
